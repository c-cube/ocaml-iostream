Module Iostream.In_buf

Buffered input stream.

type t = private {
  1. buf : bytes;
    (*

    The buffer. Do not modify directly, use fill_buffer or fill_and_get to refill it.

    *)
  2. mutable len : int;
    (*

    Length of the slice in buf.

    *)
  3. mutable off : int;
    (*

    Offset in buf.

    *)
  4. fill_buffer : bytes -> int;
    (*

    Refill buf, return new length. The newly read bytes must live in buf[0..len].

    *)
  5. close : unit -> unit;
    (*

    Close the stream. Idempotent.

    *)
}

val fill_buffer : t -> unit

fill_buffer bic ensures that bic.buf is empty only if bic.ic is empty. Always call this before looking at bic.buf.

val fill_and_get : t -> bytes * int * int

Ensure the underlying buffer is not empty (unless end-of-input was reached), and return the active slice of it as a tuple bytes, offset, len.

Postconditions: len = 0 if and only if the whole input stream has been consumed.

val consume : t -> int -> unit

consume bic n consumes n bytes from bic. Precondition: n <= get_len bic, ie. one cannot consume bytes that have not yet been obtained via fill_buffer or fill_and_get.
